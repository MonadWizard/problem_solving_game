# Pirate Lore — World Bible

## The absolute rule

**NO copyrighted IP.** No One Piece names, characters, locations, devil fruits, artwork, or thinly-veiled versions of them. Everything here is original. If a name feels borrowed, rename it.

## The world

A rookie programmer-pirate sails **the Grand Line of Problems** toward the legendary treasure: **The One Solution** — said to compile on any machine, in any language, on the first try.

- **The First Sea** — journey 1, calm-ish teaching waters (NeetCode 150).
- **The Blind Sea** — journey 2, fog everywhere; charts show no patterns (Blind 75, timed).
- **The Abyss** — a rumored third sea. Coming soon.

## Cast (original)

- **Captain Nullptr** — the player's retired-legend mentor; speaks in dry compiler metaphors. Voice of arrival stories.
- **The Compiler King** — distant antagonist whose ghost fleet enforces "eternal Wrong Answer" on the seas.
- **Bosun Segfault** — excitable first mate, delivers item drops and chest toasts.
- **Island bosses** — each island's final problem manifests as a sea-beast or rival captain named after the concept (e.g. "The Two-Headed Pointer Serpent"). Concept name must stay recognizable.

## Naming rules

- Islands: always `"<Concept> Island"` (e.g. "Sliding Window Island") — users must never memorize invented names.
- Items: `Cursed Fruits` are named `<Effect> Fruit` (Rewind, Oracle, Haste). Never "Devil Fruit".
- Currency/score: bounty in **₿** (berry-neutral "buccaneer marks").

## Tone guide

Adventure-anime energy: short lines, present tense, second person ("You drop anchor…"). Each island arrival story is 3–5 lines and must genuinely teach the island's concept through a sea metaphor (sliding window = a spyglass sweeping the horizon without re-scanning). Boss intros are one dramatic line. Completion lines award the title. Humor allowed, never snark about the learner.
